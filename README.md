# OAK-SEED

oak-seed 是一个以nestjs为基础的整合快速开发框架, 目标为开箱即用js后端整合

## 整合列表

* [X] Swagger
* [X] TypeORM
* [ ] JWT
* [ ] GraphQL

## NESTJS

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
# install Nest CLI
$ npm i -g @nestjs/cli

# install
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
